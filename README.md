# Setup

Run `$ npm install`

Modify the `npm` `start` script to include your `Slack` `API` key

Run `$ npm run start` to fire up Yo-Bot

# Usage

```
# Clean install
$ npm run clean:install

# Compile Sass
$ gulp sass

# Lint JavaScript with ESLint
$ gulp lint

# Build and Lint Sass on file change
$ gulp watch
```
